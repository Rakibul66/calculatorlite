## **📱 Full-Screen Calculator with History (Flutter + Hive)**
A **modern full-screen calculator app** built using **Flutter** with **Hive database** for storing calculation history. It features **real-time expression evaluation, a dark theme, and a smooth UI**. 🚀

---

## **🛠️ Technologies Used**
### **1️⃣ Flutter**  
- **Framework:** Cross-platform mobile development  
- **UI Toolkit:** Modern UI with `flutter_screenutil` for responsiveness  

### **2️⃣ Hive (NoSQL Local Database)**  
- **Used for:** Storing and managing calculation history  
- **Benefits:** Fast, lightweight, and easy to use  

### **3️⃣ Math Expressions (`math_expressions` Package)**  
- **Purpose:** Evaluates mathematical expressions dynamically  
- **Supports:** Basic arithmetic operations  

### **4️⃣ Date & Time Formatting (`intl` Package)**  
- **Purpose:** Formats history timestamps (`yyyy-MM-dd HH:mm:ss`)
